About Us

IoTBlock was started to bring governance and oversight of health, safety, functional integrity, and real-time industry certification status of every IoT devices, through creation of a governing public private consortium and The Universal Blockchain Database, a universal blockchain database for IoT and their users.

Our founding team includes talent from MIT, Berkeley, Stanford, ex-Google and Kodak is working with existing and next-generation RFID technology, IoTBlock will build the world’s first, decentralized Blockchain Hypercat compatible IoT device governance platform featuring an Universal IoT Blockchain Database and Hypercat REST API that interacts directly with Ethereum blockchain.

The Universal IoT Blockchain Database

Our platform will let users get paid to build a catalogue of IoT Devices by governance and integrity certification entities providing relevant information for the device.

Using this this database, users are able to identify and verify the health, safety and integrity of all IoT devices in real time that can inform its end-users whether the devices are harmful, broken, malfunctioning, tampered with, or hacked in real-time.

IoTBlock will build the world’s first, decentralized PAS212:2016 compatible IoT device governance & integrity verification platform featuring an Universal IoT Blockchain Database on Ethereum, with APIs to build an App ecosystem such as for OS deep integration, Process Automation, Supply Chain Automation, and many more.

Hypercat API Operating System Deep Integration

As an extension to our Universal IoT Blockchain Database, IoTBlock will provide Hypercat compatible API that allows Deep Integration with IoT Operating Systems to immediately halt device upon detection of malfunction, or harmful use.

The API can also be used to build applications for supply chain and business automation to avoid known harmful device manufacturers / frequently malfunctioning devices, and to source data and products from devices and manufacturers with good track record of providing products with good functional integrity.

Core Features

1) Universal IoT Blockchain Database

Blockchain database for IoT devices & their users that contains the health, safety, and real-time functional integrity certification status, with eventual focus to build out the database to cover all manufactured goods available to date.

2) Hypercat Standard for Data Delivery

Our data will use the format agreed by the British Standards Institute (BSI PAS212:2016) as the standard format for exchanging IoT data.

3) Smart Keys

Smart Keys will be used to manage ownership and access permissions for IoT devices linked to IoTBlock and allows the IoTBlock token economy to be sustainable and thriving. End users may load our tokens into the smart keys and tokens will be deducted from the key whenever tasks are performed and data is read, written, or transfered from our chains.

4) IoT Blockchain Database as a Service (DBaaS)

IoTBlock aims to adopt Database as a Service model to bring governance and Integrity in the Smart City and IoT operation, to ensure the industry thrives and starts to offer first-class service to its end-users.

Combined with the use of Smart Keys, IoTBlock aims to become the world’s first Paid IoT Hypercat Database as a Service.

In order to join IoTBlock ecosystem, end users must purchase Smart Key which serves as the user’s identity and authorization tokens for data access. Each IoT Operator has an ability to use the Smart Key to purchase a catalogue entry to feature their IoT Ecosystem in a Graph Nodal Structure. Every added information on the catalogue ecosystem incurs a fee, which is stored both on the root node (which is operated by us) and the catalogue node. The fee users pay for an information entry on the node will be split amongst the root node (us) and the catalogue creator on 50/50 basis.

5) IoTBlock REST API (For building a Mobile & Web apps)

We will have a Hypercat REST API platform that allows end users to:

  • Access our Iot device database.
  • Manage their Smart Key linked devices.
  • Manage the IoT hardware leased or purchased by them.
  • Access Mobile wallet to buy IoTBlock Smart Key tokens and refill their Smart Keys
  • Create mobile applications
  • Deep Operating System Integration to self check harmful or reported malfunctions

The API will allow end-users to easily integrate applications with the Universal Blockchain IoT Database. The usage includes, but not limited to, AI ML data for Asset Integrity Verification, reports & analytics generation, IoT catalogue graph connection overview to get a view of interconnectivity between IoT devices / sensors / scanners / smart chips within locations of interest, along with their functional integrity reporting & verification.
