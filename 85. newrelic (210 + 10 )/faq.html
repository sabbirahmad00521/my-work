What is New Relic Insights?

New Relic Insights is a real-time analytics platform that collects events directly from your live production software and transforms them into actionable insights about your business, customers, and applications. [TOP]

What is NRQL?

NRQL, New Relic Query Language, is a SQL-flavored query language used to query the New Relic Insights database. Read more on how to get started using NRQL. [TOP]

Do I need to be a developer (or know SQL) to use New Relic Insights?

SQL knowledge will help in getting started with using New Relic Insights like a pro, but it is not a required skill. New Relic Insights comes equipped with a built in tutorial to quickly become familiar with the product, pre-built example queries to experiment with, as well as an autocomplete feature to help guide the way when writing your own queries.[TOP]

What database does New Relic Insights use?

New Relic Insights is powered by a highly-distributed cloud-hosted event database with an innovative architecture that does not require indexing. The database runs on a super-cluster built to support our thousands of customersâ€”giving you access to a database that can query your big data in seconds. Learn more about New Relic Insights Infrastructure. [TOP]

What Events are collected for New Relic Insights?

Out of the box, New Relic Insights collects data on three event types from your web and mobile applications.

  • For web applications, New Relic automatically collects PageViews events, which are instrumented by New Relic's Browser Monitoring and Transaction events, which are instrumented by the New Relic APM agent (.NET, Java, Ruby, Python, PHP, Node.js).
  • For native mobile applications, New Relic automatically collects MobileSession events directly from your native mobile applications using the New Relic Mobile APM SDKs (iOS and Android).

By default, New Relic Synthetics populates Insights with a set of default attributes. Query SyntheticCheck to return metrics from one run of a particular monitor, while SyntheticRequest attributes return results from individual HTTP requests made during a check. [TOP]

Can I add additional data my app collects to New Relic Insights?

Yes, New Relic has designed adding custom attributes to be a simple process. It typically requires no more than one additional line of code to your app per attribute. Decorating your PageView and Transaction events with custom attributes is a powerful way to collect events from your software that are not collected out of the box. MobileSession Events do not currently support adding custom attributes. Learn more about how to add custom attributes and what kind of data you can add. [TOP]

Can I add external data to New Relic Insights?

Use custom events to create new event types in addition to the default PageView, Transaction, and MobileSession event types. Custom events allow you to store data in Insights that is not available during a page view, server transaction or mobile session event. You insert custom event data by submitting JSON to our HTTP endpoint. [TOP]

Can I extract data from New Relic Insights?

Exporting data from New Relic Insights is available via New Relic's API or via JSON code provided with each chart, graph and dashboard. Find out more about exporting New Relic Insights data. [TOP]

How much does New Relic Insights cost?

New Relic Insights subscription pricing is based on the number of events stored in our Real-time Analytics Platform. Customers can get started with New Relic Insights for just $250/month and store up to 75 million events! New Relic Insights scales to trillions of events. [TOP]

What is the data retention period for events I can query?

It's totally up to you. New Relic Insights has customizable data retention settings, so you can keep data in Insights for as long as you want. Insights is priced by the number events that you store, so you want to chose a retention period that meets the pricing needs of your total events stored. [TOP]
